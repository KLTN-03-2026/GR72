# Bộ tài liệu nghiệp vụ mới

Tài liệu cũ theo mô hình `expert-first` đã được loại bỏ.

Bộ tài liệu hiện tại:
- `tong-quan-he-thong.md`: tổng quan kiến trúc nghiệp vụ mới.
- `spec-chuc-nang.md`: danh sách chức năng và quy tắc nghiệp vụ chuẩn.
- `luong-nghiep-vu-moi.md`: luồng nghiệp vụ chi tiết theo vai trò.
- `chuc-nang/`: spec triển khai từng chức năng, mỗi file tương ứng một chức năng FE + BE.
- `schema-dbio.sql`: schema DB mục tiêu theo nghiệp vụ mới.
- `db-thay-doi-nghiep-vu-moi.md`: ghi chú mapping DB cũ sang DB mới.
- `roles/customer/chuc-nang-customer.md`
- `roles/expert/chuc-nang-chuyen-gia.md`
- `roles/admin/chuc-nang-admin.md`
